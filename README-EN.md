## 🎯 Overview

**WP Auto Content Pro** is a comprehensive WordPress plugin that automates content creation using artificial intelligence and automatically distributes it across major social media platforms. Ideal for blogs, news portals, niche websites, and any project that requires constant, high-quality content.

### Why WP Auto Content Pro?

- ⚡ **Full automation** — Set up once, generate content 24/7
- 🧠 **4 AI providers** with automatic failover between them
- 📱 **6 social networks** natively integrated
- 🖼️ **Automatic images** via DALL-E 3, Unsplash, or Picsum
- 🔍 **SEO optimized** with Yoast SEO and RankMath support
- 🌍 **30 languages** supported for content generation
- 📊 **Full dashboard** with statistics and detailed logs

## ✨ Features

### AI Content Generation
- Support for **OpenAI GPT-4o**, **Google Gemini**, **Anthropic Claude**, and **DeepSeek**
- **Automatic fallback** system between providers (if one fails, tries the next)
- **Automatic retry** (up to 2 attempts) on generation failure
- Complete article generation with title, content, excerpt, tags, category, and meta description
- Social media captions generated automatically
- Image generation prompts included in AI response
- **Article length** control: short (600-900), medium (900-1400), or long (1400-2200 words)

### Social Media Distribution
- **Twitter/X** — OAuth 1.0a with media upload
- **Threads** — Meta API with text and image posts
- **Instagram** — Meta Graph API v19.0 for Business accounts
- **Facebook** — Page publishing with links and photos
- **TikTok** — Content Posting API v2 with photos
- **LinkedIn** — UGC Posts API v2 with articles

### Automatic Images
- **DALL-E 3** (OpenAI) — Custom AI-generated images
- **Unsplash** — High-quality professional photos
- **Picsum** — Free fallback with no API key required
- Fallback chain: DALL-E → Unsplash → Picsum
- Automatic file type detection (JPG, PNG, WebP, GIF)
- Alt text automatically set for SEO accessibility

### Smart Scheduling
- Custom cron with intervals of 30min, 2h, 6h, or daily
- Configurable **posting window** (e.g., 08:00 to 20:00)
- **Daily post limit** (1 to 24 posts per day)
- Custom timezone support
- Transient lock to prevent concurrent executions

### Admin Dashboard
- Real-time statistics (generated posts, social shares)
- Status of all AI providers and social networks
- **"Generate Now"** button for instant content creation
- Recent activity with publication details
- Quick actions for easy navigation

### Topic Management
- Full CRUD for topics (create, edit, delete)
- **Bulk CSV import**
- Batch actions (activate, pause, delete)
- Content types: Article or Tutorial
- Per-topic frequency: hourly, every 2h, 6h, daily, or weekly
- Optional category (or let the AI decide)

### Built-in SEO
- Native compatibility with **Yoast SEO**
- Native compatibility with **RankMath**
- Meta description generated automatically by AI
- SEO title set automatically
- Tags generated by AI for each post

## 🚀 Installation

### Manual Installation

1. Download the plugin:
   ```bash
   git clone https://github.com/ybngf/WP-Auto-Content-Pro.git
   ```

2. Copy the folder to your WordPress plugins directory:
   ```
   wp-content/plugins/wp-auto-content-pro/
   ```

3. Go to **Plugins** in your WordPress admin panel

4. Activate **WP Auto Content Pro**

5. Navigate to **WP Auto Content → Dashboard** in the sidebar menu

### ZIP Installation

1. Download the ZIP file from the [GitHub repository](https://github.com/ybngf/WP-Auto-Content-Pro)
2. In WordPress, go to **Plugins → Add New → Upload Plugin**
3. Select the ZIP file and click **Install Now**
4. Activate the plugin

## ⚙️ Configuration

### Initial Setup

After activating the plugin, follow these steps:

1. **Configure AI Provider** — Go to **WP Auto Content → Settings → AI** and enter the API key for your preferred provider
2. **Add Topics** — Go to **WP Auto Content → Topics** and add the subjects you want to generate content about
3. **Enable Scheduling** — Under **Settings → Schedule**, enable automatic posting
4. **(Optional) Social Media** — Configure the credentials for your desired social networks

### Reliable Cron Tip

WP-Cron relies on site visits to execute. For more reliable scheduling, set up a real server cron job:

```bash
*/30 * * * * wget -q -O - https://yoursite.com/wp-cron.php?doing_wp_cron > /dev/null 2>&1
```

## 🧠 AI Providers

### OpenAI (GPT-4o)
- **Available models**: gpt-4o, gpt-4o-mini, gpt-4-turbo, gpt-4
- **Get API Key**: [platform.openai.com](https://platform.openai.com/api-keys)
- Also used for image generation via DALL-E 3

### Google Gemini
- **Available models**: gemini-1.5-pro, gemini-1.5-flash, gemini-pro
- **Get API Key**: [aistudio.google.com](https://aistudio.google.com/app/apikey)

### Anthropic Claude
- **Available models**: claude-opus-4-6, claude-sonnet-4-6, claude-3-opus-20240229, claude-3-sonnet-20240229
- **Get API Key**: [console.anthropic.com](https://console.anthropic.com/settings/keys)

### DeepSeek
- **Available models**: deepseek-chat, deepseek-coder
- **Get API Key**: [platform.deepseek.com](https://platform.deepseek.com/api_keys)
- Most cost-effective option with good quality

### Fallback System

If the primary provider fails, the plugin automatically tries other configured providers in this order:

1. Selected provider (primary)
2. OpenAI
3. Google Gemini
4. Anthropic Claude
5. DeepSeek

The primary provider is skipped in the fallback sequence to avoid a duplicate attempt.

## 📱 Social Media

### Twitter / X

**Authentication type**: OAuth 1.0a (User Context)

Required credentials:
- API Key (Consumer Key)
- API Secret (Consumer Secret)
- Access Token
- Access Token Secret

### Threads (Meta)

Required credentials:
- Access Token
- User ID

Text and image posts. Uses the 2-step API system (create container + publish).

### Instagram (Business)

Required credentials:
- Page Access Token
- Instagram Business Account ID

Requires a mandatory image. Uses the 3-step system (create container → wait → publish).

[Set up Business account →](https://business.facebook.com/)

### Facebook Pages

Required credentials:
- Page Access Token
- Page ID

Supports link posts (automatic preview) and photo posts.

### TikTok

Required credentials:
- Access Token

Uses Content Posting API v2. Supports photo posts.

[Developer Portal →](https://developers.tiktok.com/)

### LinkedIn

Required credentials:
- Access Token
- Author URN (auto-detected)

Publishes professional articles with link preview via UGC Posts API v2.

### Schedule Settings

| Setting            | Values                           |
|--------------------|----------------------------------|
| Posts per day       | 1 to 24                          |
| Posting window      | Start and end time (00:00-23:00) |
| Default status      | Publish, Draft, or Pending       |

The cron interval is automatically calculated based on posts per day:

| Posts/day   | Interval         |
|-------------|------------------|
| 24+         | Every 1 hour     |
| 12-23       | Every 2 hours    |
| 4-11        | Every 6 hours    |
| 1-3         | Daily            |

## 📝 Topic Management

### CSV Format for Import

```csv
"WordPress Security Tips", "article", "daily", "Security"
"How to Use WooCommerce", "tutorial", "weekly", "E-commerce"
"SEO Best Practices 2024", "article", "every_6h", "Marketing"
```

**Columns**: topic, type (`article`/`tutorial`), frequency (`hourly`/`every_2h`/`every_6h`/`daily`/`weekly`), categories

### Per-Topic Frequency

Each topic has its own generation frequency. The system automatically selects the highest-priority topic (based on last generation time and configured frequency).

## 🔔 Webhooks

Configure a webhook URL to receive notifications when a new post is published:

### Webhook Payload

```json
{
  "event": "post_published",
  "post_id": 123,
  "post_url": "https://yoursite.com/example-post/",
  "post_title": "Post Title",
  "provider": "openai",
  "timestamp": "2024-01-15T10:30:00-03:00",
  "site_url": "https://yoursite.com"
}
```

### Headers

| Header              | Description                      |
|---------------------|----------------------------------|
| `Content-Type`      | `application/json`               |
| `X-WPAC-Event`      | `post_published`                 |
| `User-Agent`        | `WPAutoContentPro/2.0.0`         |
| `X-WPAC-Signature`  | HMAC SHA-256 (if configured)     |

### Webhook Security

Set a **Webhook Secret** in advanced settings to receive an HMAC SHA-256 signature in the `X-WPAC-Signature` header, allowing you to validate notification authenticity.

## ❓ FAQ

### Does the plugin work without an API key?
No. You need at least one API key from an AI provider (OpenAI, Gemini, Claude, or DeepSeek) to generate content.

### Which AI provider is recommended?
**OpenAI GPT-4o** offers the best overall quality. **DeepSeek** is the most cost-effective option. Configure multiple providers for automatic failover.

### Is social media required?
No. Social sharing is completely optional. The plugin works perfectly by just generating WordPress posts.

### Does the plugin create categories automatically?
Yes. If the AI suggests a category that doesn't exist, the plugin creates it automatically in WordPress.

### Can I generate content manually?
Yes. Use the **"Generate Now"** button on the Dashboard to instantly generate a post, regardless of the schedule.

### What happens when I uninstall the plugin?
All plugin data is removed: database tables, options, transients, post metadata, and cron events. WordPress posts generated by the plugin are preserved.

### Is it compatible with multisite?
The plugin works on each site individually in a multisite installation.

### Can I customize the AI prompt?
The prompt is internally optimized for generating high-quality content. You control the output through the topic, content type, language, and article length settings.
